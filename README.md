## Blocked Tokens

This agent **ONLY** analyzes memecoins. The following are blocked:
- Bitcoin (BTC), Ethereum (ETH), Solana (SOL)
- All large-cap tokens (>$1B market cap)
- Stablecoins (USDT, USDC)
